# ✨ Код Судьбы — поздравления по дате рождения

Лид-магнит цифрового психолога: калькулятор, который по дате рождения определяет
**архетип человека** (сильные стороны, зоны роста) и генерирует **персональное
поздравление** на любой праздник — не банальную открытку, а текст «в самую суть».

## Как открыть

Файл `index.html` — полностью автономная страница: без сервера, без установки.

- **Локально:** просто откройте `index.html` в браузере двойным кликом.
- **В интернете (бесплатно, GitHub Pages):**
  1. В этом репозитории откройте **Settings → Pages**
  2. В разделе «Build and deployment» выберите Source: **Deploy from a branch**,
     Branch: **main**, папка **/ (root)** → Save
  3. Через 1–2 минуты сайт будет доступен по адресу
     `https://zmeyka3777-prog.github.io/Pozdravleniya/`

## Что умеет

- Форма: имя, дата рождения, пол, кем приходится, праздник (ДР, юбилей, Новый год,
  8 Марта, 23 Февраля, свадьба, профессиональный, свой вариант), обращение ты/Вы,
  5 тонов, акценты пожеланий
- Расчёт числа жизненного пути (с мастер-числами 11/22/33) → 12 архетипов
- Генератор поздравлений: суть по архетипу → сильные стороны → деликатная
  поддержка «тонкого места» → пожелания → финал; кнопка «Другой вариант»
- Фримиум: 3 бесплатных поздравления, затем окно подписки (тарифы-заглушки,
  оплата подключается позже)

## Генерация через ИИ (Kimi K2, бесплатно)

По умолчанию тексты собирает встроенный генератор (без интернета и токенов),
каждое поздравление учитывает **три числа**: день рождения, месяц и код судьбы
(сумма всех цифр даты) — поэтому у разных дат разные тексты.

Чтобы тексты писала нейросеть с нуля:
1. Зарегистрируйтесь на [openrouter.ai](https://openrouter.ai) (бесплатно)
2. Создайте ключ: **Keys → Create Key**
3. На сайте калькулятора нажмите **⚙️** (правый верхний угол), вставьте ключ,
   включите галочку «Генерировать через ИИ» → Сохранить

Модель по умолчанию — `moonshotai/kimi-k2:free` (бесплатная). Ключ хранится
только в браузере пользователя. Для массового публичного использования ИИ
(чтобы посетителям не нужен был свой ключ) потребуется мини-прокси, например
бесплатный Cloudflare Worker — это следующий шаг.
