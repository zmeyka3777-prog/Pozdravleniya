# Код Судьбы — калькулятор поздравлений по дате рождения (лид-магнит)

## Идея
Лид-магнит для рилсов цифрового психолога: пользователь вводит дату рождения человека —
получает его **архетип** (сильные стороны, зоны роста) и **персональное поздравление**,
которое «попадает в суть человека», а не банальную открытку.

## Прототип
`public/numerology.html` — полностью автономная страница (HTML+CSS+JS, без сервера).
Открывается напрямую или по адресу `/numerology.html` при запуске `npm run dev`.

### Что умеет
- **Форма:** имя, дата рождения, пол, кем приходится, праздник (ДР, юбилей, НГ, 8 Марта,
  23 Февраля, свадьба, профессиональный, свой вариант), обращение ты/Вы,
  5 тонов (тёплый, торжественный, с юмором, философский, дерзкий), акценты пожеланий.
- **Нумерология:** число жизненного пути (с мастер-числами 11/22/33), число дня рождения,
  персональный год. 12 архетипов с описаниями: суть, сильные стороны, зоны роста,
  «скрытая струна» (проблемное место, подаётся как поддержка), фирменное пожелание.
- **Генератор:** собирает поздравление из блоков (открытие → суть по архетипу →
  признание сильных сторон → деликатная поддержка слабого места → пожелания → финал),
  каждый блок варьируется по тону; кнопка «Другой вариант» перегенерирует бесплатно.
- **Фримиум:** 3 бесплатных расчёта (localStorage), потом paywall-модалка
  с тарифами-заглушками (199 ₽ пакет / 299 ₽ мес / 1990 ₽ год).
- Копирование в буфер, glassmorphism-дизайн под мистическую эстетику.

## Дорожная карта (при подтверждении идеи)
1. **MVP-паблик:** выложить страницу на отдельный домен/поддомен (Vercel/Netlify — бесплатно).
2. **Сбор лидов:** перед первым расчётом — поле «ваш Telegram/email» → база подписчиков.
3. **AI-генерация (v2):** заменить шаблоны на вызов Claude API через мини-бэкенд
   (шаблоны остаются бесплатным тиром, AI — платным: реальный расход токенов
   только у платящих). Примерная себестоимость 1 AI-поздравления ~0.3–1 ₽.
4. **Оплата:** ЮKassa / CloudPayments / Telegram Stars (если делать ботом).
5. **Telegram-бот** — альтернативный/параллельный канал: та же логика, выше конверсия из рилсов.
6. **Апселлы:** полный разбор даты (PDF), совместимость двух дат, прогноз на год.

## Технические заметки
- Страница не связана с приложением МДЛП — просто лежит в `public/`, ни один
  файл основного проекта не изменён.
- Расчёт: классическое число жизненного пути = свёртка суммы всех цифр даты
  с сохранением мастер-чисел 11/22/33.
