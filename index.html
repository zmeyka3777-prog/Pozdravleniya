      Генерировать поздравления через ИИ
      Бесплатный ключ: зарегистрируйтесь на openrouter.ai → Keys → Create Key. Модель Kimi K2 (free) не требует оплаты. Ключ хранится только в вашем браузере и никуда не передаётся, кроме OpenRouter.
      По умолчанию: moonshotai/kimi-k2:free. Другие бесплатные: deepseek/deepseek-chat-v3-0324:free, qwen/qwen3-235b-a22b:free
